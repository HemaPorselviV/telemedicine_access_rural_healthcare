 Telemedicine Access for Rural Healthcare
  Project Description
Telemedicine Access for Rural Healthcare is a web-based application developed to improve healthcare accessibility for people living in rural areas. The system enables patients to register, book appointments with doctors, and receive digital prescriptions online. Doctors can manage appointments and provide consultation details, while administrators can manage users and monitor the overall system.

Objectives
- Improve healthcare access for rural communities.
- Enable online doctor appointment booking.
- Provide digital prescriptions after consultation.
- Reduce travel time and healthcare costs.
- Maintain patient and appointment records securely.

 Installation

1. Install XAMPP.
2. Copy the project folder into:
   ```
   C:\xampp\htdocs\
   ```
3. Start Apache and MySQL in XAMPP.
4. Open phpMyAdmin and create a database named `telemedicine`.
5. Import the SQL file from the `sql` folder.
6. Open your browser and visit:
   ```
   http://localhost/telemedicine/
   ```

 Workflow

1. Patient registers and logs in.
2. Patient books an appointment.
3. Doctor views appointment requests.
4. Doctor updates the consultation and uploads a prescription.
5. Patient views the prescription.
6. Admin manages doctors, patients, and appointments.
